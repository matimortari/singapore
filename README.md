## Features

- **User Authentication:** Sign in using your Google or GitHub account.
- **Real-Time Docker Data:** View live snapshots of your Docker containers, images, and volumes.
- **Container Management:** Start, stop, and restart your Docker containers directly from the interface.

## Stack

- **Nuxt.js** with **Vue** composition API and **Nitro** server engine.
- **OAuth** authentication with Google or GitHub.
- **Prisma** for **PostgreSQL** database management.
- **Redis** for caching.
- **Pinia** for state management.
- **Zod** for schema validation.
- **TypeScript**.
- **ESLint**.
- **Tailwind CSS**.
- **Framer Motion**.
